SDI - Sistema de Diagnóstico Instantâneo

O que é o SDI?

Sistema de Diagnóstico Instantâneo (SDI) é um sistema de informação de resposta instantânea escalável para redes grandes (e pequenas). SDI provê um modo de executar scripts de diagnósticos em computadores ou qualquer outro tipo de script [como um DEE], além de criar páginas na internet contendo toda informação sumarizada.

SDI é um projeto Open Source licenciado sob a GNU General Public License v2. Foi escrito originalmente por Bruno Ribas e é mantido por Bruno Ribas, Diego Pasqualin e Vinicius Ruoso.

Como o SDI funciona?

Usando um tunnel SSH para comunicaçãom SDI obtém informação das máquinas e organiza diretamente no sistema e nas home pages

Obtendo o Código Fonte

Para obter o código fonte do SDI, você deve possuir o commando git. Então execute: 

git-clone http://git.c3sl.ufpr.br/pub/scm/sdi/sdi.git sdi/

Lista de e-mail

A lista de e-mail tem finalidade tanto para discutir questões de desenvolvimento do SDI como problemas dos usuários, além de abrigar discussões de ferramentas relacionadas ao SDI. Todos são bem vindos e podem enviar e-mails informando algum bug, pedindo novas funcionalidades, comentando algo que achou relevante sobre o SDI e ainda pode enviar patchs em cima do código para sdi@c3sl.ufpr.br. Para se inscrever na lista, mande um email vazio para sdi-subscribe@c3sl.ufpr.br. O histório das discussões da lista estão disponíveis em Listas.c3sl SDI

Você também pode visitar o canal IRC - encontre-se conosco em #c3sl no freenode.
